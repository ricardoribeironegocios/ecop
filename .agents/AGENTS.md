# Regras do Projeto

## Otimização de Imagens
Sempre que o usuário solicitar o upload de imagens ou manipulação de arquivos de foto, aplique automaticamente a conversão para o formato WebP para otimização de espaço. Isso garante que as imagens fiquem mais leves, economizando espaço no armazenamento (Supabase) e melhorando a performance do site.
